# ESLint Configuration for Class101

## Installation

### npm

```bash
npm install @class101/eslint-config --save-dev
```

### yarn

```bash
yarn add @class101/eslint-config -D
```

## Usage

In `.eslintrc`

```json
{
  "extends": "@class101/eslint-config"
}
```
